## Requisitos

Antes de ejecutar el proyecto, asegúrate de tener instalado

* MongoDB Compass
* Node.js

## Configuración

Una vez dentro de MongoDB Compass, haremos lo siguiente:

1. Crear una nueva conexión con la siguiente URL: mongodb://localhost:27017
2. Seguido de eso abriremos un powershell dentro de la carpeta donde se encuentre "conexion.js", por ejemplo: "D:\Octavo Semestre\Progra web II\Capa\Public\js" y ejecutaremos la siguiente linea

node conexion.js

2. dándonos por resultado un mensaje que dice "¡Conexión exitosa!"
